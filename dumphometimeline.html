Dump Home Timeline:

LONG FORMAT

Reads Twitter credentials directly from command line arguments.

java -jar TSAK.jar tsak -consumerKey=XXXX -consumerSecret=XXXX -accessToken=XXXX -accessSecret=XXXX dumpHomeTimeLine -o output.txt

SHORT FORMAT

Reads Twitter credentials from tsak.properties file as pointed by TSAK_CONF ENV variable.

java -jar TSAK.jar tsak dumpHomeTimeLine -o output.txt

Arguments:

XXXX:

Twitter APP and User Keys. Please refer to SETUP page to see how to generate these four keys.

-o:

Name of the output file to which the Home Timeline will be dumped.

Description:

Based on provided Credentials, this command dumps Home TimeLine (Based on Provided Twitter Keys) to a file. The number of dumped Tweets are subjected to Twitter rate limits and cannot exceeds 2700/15min.
